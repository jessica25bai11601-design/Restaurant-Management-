# Project Statement – Café Restaurant Ordering System

##  Problem Statement
Restaurants often need simple software solutions to manage customer orders efficiently.  
This project aims to create a menu-driven Python program where customers can place orders, and the system automatically calculates their bill.

##  Scope of the Project
- The system handles menu display  
- Takes orders with quantities  
- Manages invalid input  
- Calculates total price  
- Provides an order summary  
This is a small-scale ordering system ideal for cafés or training projects.

##  Target Users
- Café/restaurant staff  
- Students learning Python  
- Beginners who want to understand dictionaries and loops  
- Small businesses needing a simple console-based order system

##  High-Level Features
- Menu stored using a Python dictionary  
- Input handling using loops and validation  
- Auto calculation of order total  
- Displays final bill with itemized breakdown  
- Easy to modify and expand
