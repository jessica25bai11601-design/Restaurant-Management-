# Café Restaurant Ordering System (Python)

##  Project Overview
This is a simple Python-based console application that allows users to order food items from a predefined café menu.  
The program displays the menu, takes customer orders, calculates the total bill, and finally prints an order summary.

##  Features
- Displays a predefined café menu with item prices  
- Accepts multiple food item orders  
- Validates item names and quantities  
- Calculates total order cost  
- Shows a detailed order summary  
- User-friendly and interactive console experience  

##  Technologies Used
- *Python 3*
- Basic input/output
- Dictionary usage for storing menu items
